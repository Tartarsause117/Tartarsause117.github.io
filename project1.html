Switch Automation Scripts

This project is part of my work for the Programming for Network Engineers class.

It consists of a collection of Python scripts designed to automatically configure network switches using premade command lists

Key Features:

  • Automates switch connections.
  • Can support connection methods.
  • Designed for Cisco switches but adaptable to other vendors.

Technologies Used:

Python, Netmiko, TTKBootstrap, Difflib, Cisco IOS, VMWare Virtual Machines.

Challenges Overcome:

This project presented challenges such as understanding the netmiko syntax and its limitations. I ran into numerous issues with the program timing out due to an unexpected change in the switch name which I had not accounted for in its inital programming.
