Project 2: Steganography Tool

A simple Steganography tool that provides basic encryption.

Project Overview

The tool is a basic Steganography encryption tool that can take images and encrypt messages into them, to prevent unauthorised access an encryption key can be randomly generated by the user and used with the tool. A basic GUI is provided for ease of use.

Technologies Used

  • Python
  • Fernet (Symmetric Encryption)
  • TTKBootstrap
  • Pillow and Numpy
  • Cryptography

Key Features

  • AES Encryption via Fernet
  • Error Handling
  • Hides messages in the LSB's of an image
